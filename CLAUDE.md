# Feishu_skill - 飞书渠道销售经理数据研判助手

## 项目概述
本项目为飞书渠道销售经理专属的 AI 数据研判 Skill，基于部门每周统一导出的渠道伙伴大盘数据表，自动完成数据提炼、劣质伙伴分级筛查、全局KPI进度缺口监测、风险总结。

## 项目结构
```
Feishu_skill/
├── CLAUDE.md                          # 项目说明（本文件）
├── generate_demo_data.py              # Demo 数据生成脚本
├── skills/
│   └── lark-channel-sales/
│       ├── SKILL.md                   # Skill 主文件（核心逻辑，16.9KB）
│       └── example_template.csv       # 示例数据模板（单周参考）
└── data/                              # Demo 数据 + 实际周度数据
    ├── demo_W1_2026-04-24_to_2026-04-30.csv   # 第1周：20家基准
    ├── demo_W2_2026-05-01_to_2026-05-07.csv   # 第2周：趋势初现
    ├── demo_W3_2026-05-08_to_2026-05-14.csv   # 第3周：分化加剧
    ├── demo_W4_2026-05-15_to_2026-05-22.csv   # 第4周：趋势定型
    └── demo_all_weeks_combined.csv            # 合并总表（80行）
```

## 核心功能模块
1. **模块1：周度伙伴劣质业绩分级诊断** — 三级分级（高危/预警/观察）+ 问题总结
2. **模块1B：劣质伙伴周度变化趋势追踪** — 持续恶化/波动不稳定/改善回暖/顽固劣质
3. **模块2：全局KPI实时缺口监测** — 完成率、进度差、缺口测算、风险判定

## Demo 数据趋势模式（20家伙伴 × 4周）
| 模式 | 伙伴 | 说明 |
|------|------|------|
| 🟢 稳定优秀 | A, D, H, L, N | 持续高产出，远超均线 |
| 🔒 顽固劣质 | B, G, J, Q | 连续4周零/极低产出 |
| 🔻 持续恶化 | E, M, P | 正常→观察→预警→高危 |
| ⚡ 波动不稳定 | F, I, S | 忽好忽坏，反复横跳 |
| 🟢 改善回暖 | K, O, R | 高危→预警→观察→正常 |
| 💥 突发异常 | C, T | 前正常后突变归零 |

## 数据流
```
用户上传周度大盘表(.xlsx/.csv)
  → Agent 自动识别字段 + 清洗数据
  → 执行三级分级诊断
  → 执行KPI缺口监测
  → 输出极简结论
```

## 使用方式
1. 上传周度数据文件（或使用 `data/` 中的 demo 文件）
2. 说出指令（如「分析本周伙伴数据」）
3. Agent 按 SKILL.md 规范自动完成全流程分析

## 技术栈
- 纯数据分析，无外部依赖
- 文件解析：使用 Python（openpyxl/pandas）或直接通过对话上下文处理
- 可选飞书集成：lark-cli（需认证）
